# mi.project

mi (from 'minimal') is a collection of projects designed to be simple and minimal.

- [mi.css](./mi.css) - Minimal website cascading style sheets (CSS).
- [mi.void](./mi.void/) - Minimal void linux setup for xorg and wayland.
- [mi.nvim](./mi.nvim) - Minimal neovim configuration concentrated in a single file.
- [mi.astro](./mi.astro) - Minimal astro theme for contents.
